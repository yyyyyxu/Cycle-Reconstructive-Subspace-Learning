Cycle-Reconstructive Subspace Learning (CRSL demo)
=======================
In the file, there are multiple '*.m' files.

CRSL.m
Core codes of CRSL algorithm.

run_office_surf.m
Evaluate the results of the experiments on Office-Caltech 10 (SURF).
You can run the "run_office_surf.m" for your reference. 

run_COIL1_2.m
Evaluate the results of the experiments on COIL20.
You can run the "run_COIL1_2.m" for your reference.
